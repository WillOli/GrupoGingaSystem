# 🥋 Sistema Ginga

Plataforma desenvolvida para modernizar a gestão do Grupo Ginga, centralizando o controle de alunos, financeiro e o histórico de graduações em um só lugar.

## ✨ O que o sistema faz?
* **Gestão de Alunos:** Cadastro completo com foto e dados pessoais.
* **Controle Financeiro:** Gerenciamento prático de planos, mensalidades e status de pagamento.
* **Trilha de Graduação:** Registro histórico da evolução do capoeirista, respeitando a hierarquia oficial do grupo.

## 🛠️ Tecnologias
* **Backend:** Java + Spring Boot
* **Banco de Dados:** PostgreSQL
* **Frontend:** HTML, CSS e JavaScript
